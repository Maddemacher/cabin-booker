# cabin-booker
Moms cabin booker

NodeJs Cljs cabinbooker

## How to run

This application levrages docker-compose so deployment is very easy.

After docker and docker-compose is installed run:

- docker-compose build
- docker-compose up


## Ports used

- cabinbooker : 1337
- lumberjack : 5000
- kibana : 5601
